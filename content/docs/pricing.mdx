`hatchery-core` — the gateway, worker, trainer, and client documented on this
site — is open source under Apache-2.0 and free to self-host. There is no
license cost to run it on your own infrastructure; see
[Self-hosting](getting-started/self-hosting) and [Deployment](getting-started/deployment).

## Hosted Gateway

Pricing for a managed Hatchery gateway (prepaid usage-based billing, managed
GPU fleet, hosted auth) has not been published yet. This page is a
placeholder until the hosted platform's pricing is finalized.

For hosted access, enterprise deployment, or questions about adapting
Hatchery to a sensitive workflow, contact `contact@axolotl.ai`.
